# cardio-data-driven
Data-driven cardiovascular flow modeling: examples and opportunities

Codes and data used in the paper:
Data-driven cardiovascular flow modeling: examples and opportunities 
Amirhossein Arzani, Scott T. M. Dawson
https://arxiv.org/abs/2010.00131
